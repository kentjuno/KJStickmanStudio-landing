▶ Phân tích kênh đối thủ YouTube

Nhập link kênh hoặc thể loại nội dung — nhận danh sách kênh tương tự, phân tích đối thủ và lộ trình đăng bài đề xuất. Công cụ miễn phí, chạy hoàn toàn trên trình duyệt của bạn bằng API key của bạn — chúng tôi không thu thập bất cứ dữ liệu nào.

Chưa có API key? Xem hướng dẫn tạo miễn phí (~3 phút)
  1. Mở console.cloud.google.com và đăng nhập Google.
  2. Bấm chọn dự án ở góc trên → New Project → đặt tên (vd: "youtube-research") → Create.
  3. Vào thư viện API - YouTube Data API v3 → bấm Enable.
  4. Vào APIs & Services → Credentials → Create Credentials → API key.
  5. Sao chép key và dán vào ô trên. Miễn phí 10.000 đơn vị quota/ngày (đủ ~90 lần phân tích/ngày).
  6. Lưu ý: nếu đặt giới hạn cho key, chỉ chọn API restrictions → YouTube Data API v3. Nếu đặt "HTTP referrers" thì thêm stickman.kentjuno.com vào danh sách cho phép.
Nhập link/@handle của một kênh để tìm kênh tương tự, hoặc nhập từ khóa thể loại.
